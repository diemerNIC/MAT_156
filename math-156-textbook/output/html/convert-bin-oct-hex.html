Section 1.5 Converting between Binary, Octal, and Hexadecimal

Subsection 1.5.1 Converting Between Binary and Octal

Let's first count from zero to 7 in both octal and binary.

octal binary
\(0_8\) \(0_2\)
\(1_8\) \(1_2\)
\(2_8\) \(10_2\)
\(3_8\) \(11_2\)
\(4_8\) \(100_2\)
\(5_8\) \(101_2\)
\(6_8\) \(110_2\)
\(7_8\) \(111_2\)

If we add leading zeros where necessary to bring all binary numbers to three digits, we can see that

\begin{align*} 6_8 \amp = 110_2\\ 3_8 \amp = 011_2\\ 4_8 \amp = 100_2 \end{align*}

The reason we are doing this is that if we look at a number expressed both in octal and binary, such as \(634_8=110011100_2\text{,}\) we can see an interesting pattern. If the binary number is split into groups of three digits, starting from the right-hand-side,

\begin{align*} 634_8 \amp = 110011100_2\\ \amp = 110\ 011\ 110_2\\ \amp = \overbrace{110}^{6}\overbrace{011}^{3}\overbrace{100}^{4} \end{align*}

then we can see that each group of three digits is equal to the corresponding octal number in that place.

Let's use this observation to convert from octal to binary.

Example 1.5.1.
Convert the following octal numbers to binary:
  1. \(\displaystyle \ 1110_2\)
  2. \(\displaystyle \ 1010000100_2\)
Solution
  1. \(\ \)For each digit in the octal number, write the corresponding three-digit binary number (includeing any leading zeros as necessary). Then put the groups of three all together and, optionally, drop any leading zeros.
    \begin{align*} 23_8 \amp = \overbrace{010}^2\overbrace{011}^3\\ \amp = 010\ 011\\ \amp = 10011_2,\text{ dropping the leading zero} \end{align*}
  2. \begin{align*} 671_8 \amp = \overbrace{110}^6\overbrace{111}^7\overbrace{001}^1\\ \amp = 110\ 111\ 001\\ \amp = 110111001_2 \end{align*}

We can use a similar technique to convert from binary to octal.

Example 1.5.2.
Convert the following binary numbers to octal.
  1. \(\displaystyle \ 11110_2\)
  2. \(\displaystyle \ 1010000100_2\)
Solution
First, group the binary digits into threes, starting from the right-hand side. The last group might need leading zeros to be added to make a group of three. Then rewrite each set of three into the corresponding digit in octal.
  1. \(\displaystyle \ 11110_2=011\ 110=\overbrace{011}^3\overbrace{110}^6=36_8\)
  2. \(\displaystyle \ 1010000100_2=001\ 010\ 000\ 100=\overbrace{001}^1\overbrace{010}^2\overbrace{000}^0\overbrace{100}^4=1204_8\)

Subsection 1.5.2 Converting Between Binary and Hexadecimal

As before, let's first count from zero to fifteen in both hexadecimal and binary.

hexadecimal binary
\(0_{16}\) \(0_2\)
\(1_{16}\) \(1_2\)
\(2_{16}\) \(10_2\)
\(3_{16}\) \(11_2\)
\(4_{16}\) \(100_2\)
\(5_{16}\) \(101_2\)
\(6_{16}\) \(110_2\)
\(7_{16}\) \(111_2\)
hexadecimal binary
\(8_{16}\) \(1000_2\)
\(9_{16}\) \(1001_2\)
\(A_{16}\) \(1010_2\)
\(B_{16}\) \(1011_2\)
\(C_{16}\) \(1100_2\)
\(D_{16}\) \(1101_2\)
\(E_{16}\) \(1110_2\)
\(F_{16}\) \(1111_2\)

If we add leading zeros where necessary to write all binary numbers with four digits, we can see that, for example,

\begin{align*} 9_{16} \amp = 1001_2\\ D_{16} \amp = 1101_2\\ 2_{16} \amp = 0010_2 \end{align*}

Now consider the number \(9D2_{16}\text{.}\) To convert this to binary, write each hexadecimal digit in 4-digit binary (padding with leading zeros as necessary, as shown above) from left to right:

\begin{align*} 9D2_{16} \amp = \overbrace{1001}^9\overbrace{1101}^D\overbrace{0010}^2\\ \amp = 100111010010_2 \end{align*}

Let's work through more examples.

Example 1.5.3.
Convert the following hexadecimal numbers to binary.
  1. \(\displaystyle \ 3A_{16}\)
  2. \(\displaystyle \ F02C_{16}\)
Solution
  1. \begin{align*} 3A_{16} \amp = \overbrace{0011}^3\overbrace{1010}^A\\ \amp = 111010_2,\text{ dropping the leading zeros} \end{align*}
  2. \begin{align*} F02C_{16} \amp = \overbrace{1111}^F\overbrace{0000}^0\overbrace{0010}^2\overbrace{1100}^C\\ \amp = 1111000000101100_2 \end{align*}
Example 1.5.4.
Convert the following binary numbers to hexadecimal.
  1. \(\displaystyle \ 11110_2\)
  2. \(\displaystyle \ 1010001111_2\)
Solution
First, group the binary digits into sets of four, starting from the right-hand side. Then rewrite each set of four into the corresponding hexadecimal digit.
  1. \(\displaystyle \ 11110_2 = 0001\ 1110 = \overbrace{0001}^1\overbrace{1110}^{14}=1E_{16}\)
  2. \(\displaystyle \ 1010001111_2 = 0010\ 1000\ 1111=\overbrace{0010}^2\overbrace{1000}^8\overbrace{1111}^{15}=28F_{16}\)

Padding the left-most group of four with leading zeros is done for clarity, rather than for necessity. You can skip that part of the process and instead write, for example,

\begin{equation*} 11110_2=\overbrace{1}^1\overbrace{1110}^{14}=1E_{16} \end{equation*}

Subsection 1.5.3 Converting Between Octal and Hexadecimal

The fastest way to do this is to convert into binary first, then regroup the binary digits.

Example 1.5.5.
Convert the following octal numbers to hexadecimal.
  1. \(\displaystyle \ 72_8\)
  2. \(\displaystyle \ 333_8\)
Solution
  1. \begin{align*} 72_8 \amp = \overbrace{111}^7\overbrace{010}^2\\ \amp = 111010_2\\ \amp = 11\ 1010\\ \amp = \overbrace{11}^3\overbrace{1010}^{A}\\ \amp = 3A_{16} \end{align*}
  2. \begin{align*} 333_8 \amp =\overbrace{011}^3\overbrace{011}^3\overbrace{011}^3\\ \amp = 011\ 011\ 011\\ \amp = 11011011_2\\ \amp = 1101\ 1011\\ \amp = \overbrace{1101}^{D}\overbrace{1011}^{B}\\ \amp = DB_{16} \end{align*}
Example 1.5.6.
Convert the following hexadecimal numbers to octal.
  1. \(\displaystyle \ 9A_{16}\)
  2. \(\displaystyle \ 4E59_{16}\)
Solution
  1. \begin{align*} 9A_{16} \amp = \overbrace{1001}^9\overbrace{1010}^A\\ \amp = 10011010_2\\ \amp = 10\ 011\ 010\\ \amp = \overbrace{10}^2\overbrace{011}^3\overbrace{010}^2\\ \amp = 232_8 \end{align*}
  2. \begin{align*} 4E59_{16} \amp = \overbrace{0100}^4\overbrace{1110}^E\overbrace{0101}^5\overbrace{1001}^9\\ \amp = 0100\ 1110\ 0101\ 1001\\ \amp = 100111001011001_2\\ \amp = 100\ 111\ 001\ 011\ 001\\ \amp = \overbrace{100}^4\overbrace{111}^7\overbrace{001}^1\overbrace{011}^3\overbrace{001}^1\\ \amp = 47131_8 \end{align*}

Subsection 1.5.4 Converting Non-integer Numbers

Converting non-integer numbers between binary, octal, and hexadecimal works in the way you'd expect.

Example 1.5.7.
Convert the following octal numbers to binary.
  1. \(\displaystyle \ 0.32_8\)
  2. \(\displaystyle \ 6.71_8\)
Solution

  1. For each digit in the octal number, write the corresponding three-digit binary number (with leading zeros if necessary). Then put the groups of three all together and, optionally, drop any leading zeros to the left of the radix point and any trailing zeros to the right of the radix point.

    \begin{align*} 0.32_8 \amp = 0.\overbrace{011}^3\overbrace{010}^2\\ \amp = 0.011\ 010\\ \amp = 0.01101_2,\text{ dropping the trailing zero} \end{align*}
  2. \begin{align*} 6.71_8 \amp = \overbrace{110}^6.\overbrace{111}^7\overbrace{001}^1\\ \amp = 110.111\ 001\\ \amp = 110.111001_2 \end{align*}
Example 1.5.8.
Convert the following binary numbers to octal:
  1. \(\displaystyle \ 11.11_2\)
  2. \(\displaystyle \ 1010.000101_2\)
Solution

First, group the binary digits into threes, starting from the radix point, adding extra leading and/or trailing zeros as appropriate. Then rewrite each set of three as the corresponding octal digit.

  1. \(\displaystyle \ 11.11_2 = 011.110=\overbrace{011}^3.\overbrace{110}^6=3.6_8\)
  2. \(\displaystyle \ 1010.000101_2=1\ 010.000\ 101=\overbrace{1}^1\overbrace{010}^2.\overbrace{000}^0\overbrace{101}^5=12.05_8\)
Example 1.5.9.
Convert the following hexadecimal numbers to binary.
  1. \(\displaystyle \ 3.A_{16}\)
  2. \(\displaystyle \ F.02C_{16}\)
Solution
  1. \begin{align*} 3.A_{16} \amp = \overbrace{0011}^3.\overbrace{1010}^A\\ \amp = 11.101_2,\text{ dropping the leading and trailing zeros} \end{align*}
  2. \begin{align*} F.02C_{16} \amp = \overbrace{1111}^F.\overbrace{0000}^0\overbrace{0010}^2\overbrace{1100}^C\\ \amp = 1111.0000 0010 11_2 \end{align*}

We use the same techniques as before to convert non-integer numbers between octal and hexadecimal: convert to binary as an intermediate step.

Example 1.5.10.
Convert the following hexadecimal numbers to octal.
  1. \(\displaystyle \ 9.A_{16}\)
  2. \(\displaystyle \ 4.E59_{16}\)
Solution
  1. \begin{align*} 9A_{16} \amp = \overbrace{1001}^9.\overbrace{1010}^A\\ \amp = 1001.1010_2\\ \amp = 1\ 001.101\ 0\\ \amp = \overbrace{1}^1\overbrace{001}^1.\overbrace{101}^5\\ \amp = 11.5_8 \end{align*}
  2. \begin{align*} 4.E59_{16} \amp = \overbrace{0100}^4.\overbrace{1110}^E\overbrace{0101}^5\overbrace{1001}^9\\ \amp = 0100.1110\ 0101\ 1001\\ \amp = 100.111001011001_2\\ \amp = 100.111\ 001\ 011\ 001\\ \amp = \overbrace{100}^4.\overbrace{111}^7\overbrace{001}^1\overbrace{011}^3\overbrace{001}^1\\ \amp = 4.7131_8 \end{align*}
