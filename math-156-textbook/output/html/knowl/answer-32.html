Answer 3.1.4.1.

\begin{align*} a_n \amp = 4n + 7, \text{so}\\ a_0 \amp = 4\times0 + 7 = 7\\ a_1 \amp = 4\times1 + 7 = 11\\ a_2 \amp = 4\times2 + 7 = 15\\ a_3 \amp = 4\times3 + 7 = 19 \end{align*}
The first four terms are then 7, 11, 15, and 19. This is the same sequence that was given as part (a) in the first example of this section.
