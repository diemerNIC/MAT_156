Answer 3.2.7.1.

Let's use a starting index of one. So \(a_1\) is \(-37\) and \(d\) is \(+9\text{.}\) Then we want to find the value of \(n\) for which \(a_n\) equals 404.
\begin{align*} a_n \amp = a_m + \left( n - m \right)d\\ \amp = a_1 + \left(n - 1 \right)d\\ 404 \amp = - 37 + \left( n - 1 \right)9\\ 441 \amp = 9\left( n - 1 \right)\\ 49 \amp = n - 1\\ n \amp = 50 \end{align*}
The fiftieth term is 404.
