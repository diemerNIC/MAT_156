This is just the same sequence as before, with first term \(a_m = 2\) and common difference \(d = 8\text{.}\) In order to use our previous formula, however, we need to calculate the last term \(a_n\text{.}\) If we start with an index of one, then the fortieth term will be \(a_{40}\text{,}\) and we will need that value to calculate \(S_{40}\text{.}\)
\begin{align*} a_n \amp = a_m + \left( n - m \right)d\\ \amp = a_1 + \left( n - 1 \right)d\\ a_{40} \amp = 2 + 39 \times 8\\ \amp = 314 \end{align*}
So,
\begin{align*} S_k \amp = \frac{k}{2}\left( a_m + a_n \right)\\ S_{40} \amp = \frac{40}{2}\left( 2 + 314 \right)\\ \amp = 20 \times 316\\ \amp = 6320 \end{align*}
The sum of the first forty terms is 6320. (Much easier than writing out the first forty terms and adding them up!)
