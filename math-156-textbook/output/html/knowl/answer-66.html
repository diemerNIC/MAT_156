Answer 3.3.16.1.

\begin{align*} 0.\overline {18} \amp = 0.1818181818\ldots\\ \amp = 0.18 + 0.0018 + 0.000018 + \ldots \end{align*}
But this is just the sum of an infinite geometric series with \(a_m = \frac{18}{100}\) and \(r = \frac{1}{100}\text{.}\) Then
\begin{equation*} {S_\infty } = \frac{{{a_m}}}{{1 - r}} = \frac{{\frac{{18}}{{100}}}}{{1 - \frac{1}{{100}}}} = \frac{{\frac{{18}}{{100}}}}{{\frac{{99}}{{100}}}} = \frac{{18}}{{100}} \times \frac{{100}}{{99}} = \frac{{18}}{{99}} = \frac{2}{{11}} \end{equation*}
So \(0.\overline {18} = 2/11\text{.}\)
