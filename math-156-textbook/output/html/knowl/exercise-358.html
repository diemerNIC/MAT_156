\(\ \)Given the conditional statement, If Bossy is mooing, she must be a cow, which of the four following statements is the contrapositive (\(\contrapositive\))?

  1. If Bossy is not a cow, she is not mooing.

  2. If Bossy is a cow, then she is mooing.

  3. If Bossy is mooing, then she must be a cow.

  4. if Bossy is not mooing, then she must not be a cow.
