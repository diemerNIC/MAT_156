Another thing to note is what happens when we try to write the decimal number 16 in base 4. The previous number, 15, is written as \(33_4\text{.}\) When you add one to \(33_4\text{,}\) the three in the right-hand column increments to four. The base is four, however, and so we write a zero instead and add one to the next column to the left. Since the digit is that column is also a three, we write a zero and write a one in the next column to the left. This yields \(16=100_4\text{.}\)
