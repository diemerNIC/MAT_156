When you are doing arithmetic, to evaluate the expression
\begin{equation*} 4+3\times 2^2 \end{equation*}
you need to know which operation (addition, multiplication, exponentiation) should come first. In the same way, there is an order of operations, or precedence, in logic:
  • negation, \(\sim\text{,}\) is done first
  • and, \({\wedge}\text{,}\) is done next
  • or, \({\vee}\text{,}\) is done last
and brackets () override the default precedence. If you like, you can think of not like exponents, and like multiplication, and or like addition (more on this later).
