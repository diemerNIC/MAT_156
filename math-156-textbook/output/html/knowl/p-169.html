Let's now look at how to write the truth table for \(p\myand 1\text{,}\) where 1 means a statement that is always true. (Be careful! The number 1 is a constant, not a variable. It never takes the value of zero.) The table would look like:

\(p\) \(1\) \(p\myand 1\)
\(0\) \(1\) \(0\)
\(1\) \(1\) \(1\)
Notice that the last column looks like the first, so \(p\myand 1\) has the same values as \(p\text{.}\) We say, then, that \(p\myand 1\) is logically equivalent to \(p\text{.}\) The topic of logical equivalence will be examined in more detail later on.
