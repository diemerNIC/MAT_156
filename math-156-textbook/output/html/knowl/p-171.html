To negate an expession, we use the same idea and switch the value of each entry in the column for that expression, whether that expression is simple or compound. For example, here is the truth table for \(\sim\!(p\wedge 1)\text{:}\)

\(p\) 1 \(p\wedge 1\) \(\sim\!(p\wedge 1)\)
0 1 0 1
1 1 1 0
The fourth column is the negation of the third column: the values in the fourth column are obtained by applying the switches \(0\to 1\) and \(1\to 0\) to the values in the third column.
