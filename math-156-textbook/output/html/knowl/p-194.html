The order of Boolean operations is indicated in the following examples:
  • \(AB+C\qquad\) First evaluate \(AB\text{,}\) then use the result as the first variable in the or operation with \(C\)
  • \(A+BC\qquad\) First evaluate \(BC\text{,}\) then use that result as the second variable in the or operation with \(A\text{.}\)
  • \(\overline{A}{}B\qquad\) First evaluate the negation of \(A\text{,}\) then use that result in the and operation with \(B\text{.}\)
  • \(\overline{A+B}\qquad\) First evaluate \(A+B\text{,}\) then negate the result. Note the implied grouping of \(A+B\text{.}\)
  • \(A(B+C)\qquad\) First evaluate the grouped expression, then use the result in the and operation with \(A\text{.}\)
