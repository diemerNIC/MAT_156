Earlier, we expanded 179 as
\begin{equation*} 179=1\times 10^2+7\times 10^1+9\times 10^0. \end{equation*}
We can expand in octal in the same way, using powers of 8 instead of powers of 10:
\begin{align*} 263_8 \amp = 2\times 8^2+6\times 8^1+3\times 8^0\\ \amp = 2\times 64+6\times 8+3\times 1\\ \amp = 128+48+3\\ \amp = 179 \end{align*}
So we see that 179 is equivalent to \(263_8\text{.}\)
