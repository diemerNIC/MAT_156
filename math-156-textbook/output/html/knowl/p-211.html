You can replace the logic symbols in this table with the corresponding Boolean algebra notation. From this, we can deduce the following identity laws:

Identity Laws.

Logic
\begin{align*} p{\wedge} 0 \amp\Leftrightarrow 0\\ p{\vee} 0\amp\Leftrightarrow p\\ p{\wedge} 1\amp\Leftrightarrow p\\ p{\vee} 1\amp\Leftrightarrow 1 \end{align*}
Boolean
\begin{align*} A\cdot 0\amp = 0\\ A+0\amp = A\\ A\cdot 1\amp = A\\ A+1\amp = 1 \end{align*}

It is interesting to note that the first three Boolean identity laws look like laws of normal arithmetic with numbers. The fourth law does not resemble any arithmetic law. This is a clue that Boolean algebra is not equivalent to the algebra associated with the real numbers.
