From this table we can see that

\begin{equation*} \sim\!(p{\wedge} q)\Leftrightarrow\sim\!{p}\,{\vee}\sim\!{q}. \end{equation*}

We can construct a similar table to show that

\begin{equation*} \sim\!(p{\vee} q)\Leftrightarrow\sim\!{p}\,{\wedge}\sim\!{q}. \end{equation*}

These are called De Morgan's Laws. Constructing the tables using Boolean algebra notation yields similar statements.
