If \(\conditional\) is true, can we conclude that \(\converse\) is also true? The statment \(\converse\) is called the converse of the statement \(\conditional\text{.}\) Let's use our previous example again:

Original statement: If Barney is a dog, then Barney has four legs.

Converse: If Barney has four legs, then Barney is a dog.
