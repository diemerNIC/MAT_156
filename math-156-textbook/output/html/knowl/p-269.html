Let's now consider the conditional If the temperature outside is below \(0^{\circ}\text{C}\text{,}\) then it is freezing outside. If I were to use this sentence in everyday English, I probably mean If the temperature outside is below \(0^{\circ}\text{C}\text{,}\) then it is freezing outide and if the temperature outside is not below \(0^{\circ}\text{C}\text{,}\) then it is not freezing outside. So we could probably do with a new kind of connective that means If \(p\) then \(q\) and if \(\sim\!{p}{}\) then \(\sim\!{q}{}\). The new connective is called the biconditional, and is written \(\biconditional\)
