There are a number of ways to specify the biconditional in English:

  1. If and only if \(p\text{,}\) then \(q\text{.}\)

  2. \(p\) if and only if \(q\text{.}\)

  3. If \(p\text{,}\) then \(q\text{,}\) and vice versa.

  4. If \(p\text{,}\) then \(q\text{,}\) and if \(\sim\!{p}{}\) then \(\sim\!{q}{}\text{.}\)

We'll mostly be using the first two in this list.
