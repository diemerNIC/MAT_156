Here’s how to fill in the columns for any logical expression containing the conditional connective (\({\rightarrow}\)). Let’s call the propositions first and second so we don’t get confused with \(p\) and \(q\text{.}\) For the conditional statement first\({\rightarrow}\) second, it will be be true for all cases except when the first is true and the second is false. So for \(\converse\text{,}\) look for the row in which \(q\) is true and (rows 2 and 4) \(p\) is false (rows 1 and 2). Then \(q\) is true and \(p\) is false only for row 2. Therefore enter F in the second row, and enter \(T\) in the remaining rows of the column for \(\converse\text{.}\) This table shows that the conditional and the converse are not logically equivalent.
