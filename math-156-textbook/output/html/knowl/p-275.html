Let's examine this in more detail. Recall that if \(\conditional\) is true and \(q\) is true, we cannot conclude anything about \(p\text{.}\) Now consider the following piece of pseudocode:

              x=4
              y=0
              if x > 3 then y = 5
              print "y = ", y

The output will be "y=5".
