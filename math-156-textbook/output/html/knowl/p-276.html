Will the following pseudocode produce the same output?

              x=2
              y=5
              if x > 3 then y=5
              print "y = ",y

It will. The conditional statement did not change the value of \(y\text{,}\) but the value was set to 5 initially and so the output will be "y=5". Once again, knowing that \(q\) is true does not allow us to conclude anything about \(p\) from the conditional \(\conditional\text{.}\) However, if the output was "y=4" or any other value not equal to 5, we can draw the conclusion that \(x\) was not greater than 3. if \(\conditional\) is true and \(q\) is false, we know with certainty that \(p\) is false also.
