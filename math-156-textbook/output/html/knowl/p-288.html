  2. \(p\) if and only if \(q\text{.}\)
  3. If \(p\text{,}\) then \(q\text{,}\) and vice versa.
  4. If \(p\text{,}\) then \(q\text{,}\) and if \(\sim\!{p}{}\) then \(\sim\!{q}{}\text{.}\)
