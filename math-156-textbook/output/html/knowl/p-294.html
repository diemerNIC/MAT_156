For special cases, the if-then-else construction can yield more information. Consider the following:
              if x > 3 then
                y = 5
              else
                y = 7
              print "y = ",y
Since this piece of pseudocode assigns different values (5 or 7) to the same variable \(y\text{,}\) finding out the resulting value of \(y\) will determine whether \(x\) was greater than 3. In this special case, the if-then behaves like the biconditional: if \(y=5\) then you know that \(x \gt 3\text{,}\) and if \(y\ne 5\text{,}\) then \(x\leq 3\text{.}\)
