Also, the total number of terms in \(a_0, a_1, a_2, a_3, \ldots, a_n\) is given by
\begin{align*} k \amp = n - m + 1\\ \amp = n - 0 + 1\\ \amp =n+1 \end{align*}
so \(k\text{,}\) the “count” of the term is no longer equal to \(n\text{,}\) the index of the final term. So be warned: if you are not careful with this convention, you are likely to make a type of mistake which programmers commonly call an “off-by-one” error.
