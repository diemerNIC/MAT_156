Let’s look at a number in base 4, so we’re not getting confused with all the zeros and ones. If we consider
\begin{equation*} 20.31_4 \end{equation*}
we see that the two digits to the left of the radix point, 2 and 0, make up the integer part, while the 3 and the 1 make up the fractional part. Using positional notation, the 3 is in the fourths place while the 1 is in the sixteenths place. we have seen before that \(20_4=2\times4^1+4\times4^0\text{,}\) so now
\begin{equation*} 20.31_4=2\times4^1+0\times4^0+3\times4^{-1}+1\times4^{-2} \end{equation*}
or, if you prefer,
\begin{equation*} 20.31_4=2\times4+0+\frac{3}{4}+\frac{1}{4^2}. \end{equation*}
