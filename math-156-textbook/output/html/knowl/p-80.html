If we add leading zeros where necessary to bring all binary numbers to three digits, we can see that

\begin{align*} 6_8 \amp = 110_2\\ 3_8 \amp = 011_2\\ 4_8 \amp = 100_2 \end{align*}

The reason we are doing this is that if we look at a number expressed both in octal and binary, such as \(634_8=110011100_2\text{,}\) we can see an interesting pattern. If the binary number is split into groups of three digits, starting from the right-hand-side,

\begin{align*} 634_8 \amp = 110011100_2\\ \amp = 110\ 011\ 110_2\\ \amp = \overbrace{110}^{6}\overbrace{011}^{3}\overbrace{100}^{4} \end{align*}

then we can see that each group of three digits is equal to the corresponding octal number in that place.
