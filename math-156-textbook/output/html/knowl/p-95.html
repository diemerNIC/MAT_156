Taking this idea one step further, we can consider she in the third example to behave like a variable, and whether the full statement "she was late" is true or false must depend on what the value of the variable she is. Similarly, in programming it is very common to evaluate the value (true or false) of propositions like \(x=3\) or \(y \lt 5\) in statements like

              if x=3 then print ''Hello World''

provided that, like she/Laura, the value of \(x\) has been defined previously.
