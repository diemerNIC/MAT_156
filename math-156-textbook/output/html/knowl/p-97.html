The negation of any proposition \(p\) is called not p and is written using the tilde symbol, \(\sim\!{p}\text{.}\) The tilde can be found on a standard keyboard as the shifted key to the left of the 1. You may also see negations in logic written using this symbol, \(\neg p\text{,}\) or using a bar or overline, \(\overline{p}\text{.}\) In computing, you will also see \(!p\) as the negation. We will use the \(\sim\!{p}\) because it uses a character on a conventional keyboard, so it is easier to type.
