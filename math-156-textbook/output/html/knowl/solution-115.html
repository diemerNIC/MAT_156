Solution 1.5.1.1.

  1. \(\ \)For each digit in the octal number, write the corresponding three-digit binary number (includeing any leading zeros as necessary). Then put the groups of three all together and, optionally, drop any leading zeros.
    \begin{align*} 23_8 \amp = \overbrace{010}^2\overbrace{011}^3\\ \amp = 010\ 011\\ \amp = 10011_2,\text{ dropping the leading zero} \end{align*}
  2. \begin{align*} 671_8 \amp = \overbrace{110}^6\overbrace{111}^7\overbrace{001}^1\\ \amp = 110\ 111\ 001\\ \amp = 110111001_2 \end{align*}
