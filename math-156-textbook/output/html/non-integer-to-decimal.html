Section 1.3 Converting Non-Integer Numbers to Decimal

Subsection 1.3.1 Review of the Decimal System for Non-integers

Let's once again review the decimal system, but this time we will consider non-integer numbers. Recall that integers are numbers that can be written without a fractional part, like 5, -3, and 0. To write non-integer numbers as a decimal, we again use positional notation, where the fractional part is to the right of the decimal point.

For example, consider the base-10 number

\begin{equation*} 8.76 \end{equation*}

The dot is called the decimal point (some cultures use a comma rather than a dot). The digit to the immediate left (8) is in the "ones" place, the first digit to the right (7) is in the "tenths" place, and the second digit to the right (6) is in the "hundredths" place:

\begin{equation*} 8.76=8+\frac{7}{10}+\frac{6}{100} \end{equation*}

Recall that \(\tfrac{1}{10}\) can be written as \(10^{-1}\text{,}\) we can write

\begin{equation*} 8.76=8\times10^0+7\times10^{-1}+6\times10^{-2}, \end{equation*}

or, equivalently,

\begin{equation*} 8.76=8+0.7+0.06 \end{equation*}

This may seem redundant, but this representation will come in handy when considering bases other than ten.

Example 1.3.1.
In the decimal number 38.6, state which digit is in the
  1. \(\ \)tens place
  2. \(\ \)ones place
  3. \(\ \)tenths place
Answer
  1. \(\ \)3
  2. \(\ \)8, since it is to the left of the decimal point
  3. \(\ \)6
Example 1.3.2.
In the decimal number 2.4608, in what place are the following digits?
  1. \(\ \)6
  2. \(\ \)2
  3. \(\ \)0
  4. \(\ \)8
  5. \(\ \)4
Answer
  1. \(\ \)the hundredths place
  2. \(\ \)the ones place
  3. \(\ \)the thousandths place
  4. \(\ \)the ten thousandths place
  5. \(\ \)the tenths place

Subsection 1.3.2 Non-Integers in Bases Other Than Ten

When we consider a number such as

\begin{equation*} 10.011_2 \end{equation*}

we immediately run into a naming problem. We can no longer call the dot the decimal point as this is not a decimal number. For this particular example, we can call the dot the binary point, and in general the dot is called the radix point.

Now, the digits to the left of the binary point make up the integer part of the number as before, and the digits to the right make up the fractional part.

Let's look at a number in base 4, so we're not getting confused with all the zeros and ones. If we consider

\begin{equation*} 20.31_4 \end{equation*}

we see that the two digits to the left of the radix point, 2 and 0, make up the integer part, while the 3 and the 1 make up the fractional part. Using positional notation, the 3 is in the fourths place while the 1 is in the sixteenths place. we have seen before that \(20_4=2\times4^1+4\times4^0\text{,}\) so now

\begin{equation*} 20.31_4=2\times4^1+0\times4^0+3\times4^{-1}+1\times4^{-2} \end{equation*}

or, if you prefer,

\begin{equation*} 20.31_4=2\times4+0+\frac{3}{4}+\frac{1}{4^2}. \end{equation*}

Writing these numbers in their decimal equivalents, we see that

\begin{align*} 20.31_4 \amp =8+0+0.75+0.0625\\ \amp = 8.8125_{10} \end{align*}

Similarly,

\begin{align*} F2.B9_{16} \amp =15\times16^1+2\times16^0+11\times16^{-1}+9\times16^{-2}\\ \amp = 15\times16+2\times1+\frac{11}{16}+\frac{9}{16^2}\\ \amp = 240+2+0.6875+0.035156\\ \amp = 242.72265625 \end{align*}

and we can conlude that \(F2.B9_{16}=242.72265625_{10}\)

Example 1.3.3.
In the number \(30.1C_{16}\) state which digit is in the
  1. \(\ \)sixteens place
  2. \(\ \)sixteenths place
Answer
  1. \(\ \)3
  2. \(\ \)1
Example 1.3.4.
Write the following numbers in expanded form, and then use the expanded form to determine the decimal form of the number. If appropriate, round your answer to three decimal places.
  1. \(\displaystyle \ 101.011_2\)
  2. \(\displaystyle \ 12.17_8\)
  3. \(\displaystyle \ 0.FE_{16}\)
  4. \(\displaystyle 2132.43_5\)
Answer
  1. \(\displaystyle \ 101.011_2=1\times2^2+0\times2^1+1\times2^0+0\times2^{-1}+1\times2^{-2}+1\times2^{-3}=5.375\)
  2. \(\displaystyle \ 12.17_8=1\times8^1+2\times8^0+1\times8^{-1}+7\times8^{-2}=10.234\)
  3. \(\displaystyle \ 0.FE_{16}=0\times16^0+15\times16^{-1}+14\times16^{-2}=0.992\)
  4. \(\displaystyle \ 2132.43_5=2\times5^3+1\times5^2+3\times5^1+2\times5^0+4\times5^{-1}+3\times5^{-2}=292.92\)
