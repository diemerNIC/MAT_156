    Section 3.2 Arithmetic Sequences and Series

    Subsection 3.2.1 Arithmetic Sequences

    Let‚Äôs start out with a definition:

    Definition 3.2.1.

    An arithmetic sequence is a sequence in which the next term is found by adding a constant (the common difference \(\ d\)) to the previous term.
    Here are some examples of arithmetic sequences:
    1. \(\displaystyle 7, 11, 15, 19, \ldots \)
    2. \(\displaystyle 11, 4, -3, -10, \ldots -59\)
    3. \(\displaystyle 12, 12.3, 12.6, 12.9, \ldots\)
    The first one has a common difference of 4, the second \(-7\text{,}\) and the third 0.3. Note that in each of them, we can find the common difference \(d\) by taking any term and subtracting the previous term from it.

    Example 3.2.2.

    For the following sequences, state whether each of them is arithmetic.
    1. \(\displaystyle -3, -10, -17, -24, \ldots\)
    2. \(\displaystyle 4, 5, 7, 10, \ldots\)
    3. \(\displaystyle 2, 4, 8, 16, \ldots\)
    4. \(\displaystyle \frac{1}{2},\frac{1}{3},\frac{1}{4},\frac{1}{5},\ldots \frac{1}{20}\)
    Answer.
    1. Yes, because the common difference \(d\) is \(-7\text{.}\)
    2. No, because you‚Äôre not adding the same number each time.
    3. No, because you‚Äôre multiplying by 2 to get the next term, not adding.
    4. No, because the difference between each pair of terms is different.
    Again, you can define an arithmetic sequence in one of three ways: by listing the terms, by giving a recursive definition, or by giving a general definition.

    Subsection 3.2.2 Recursive Definitions for Arithmetic Sequences

    Let‚Äôs first look at an example:

    Example 3.2.3.

    Give a recursive definition for the sequence \(2, 10, 18, 26, \ldots\text{.}\)
    Answer.
    Recall that a recursive definition has two parts: listing the first term and giving the pattern. In this case, the pattern is adding \(d = 8\) to the previous term to get the next term. We can start our index anywhere, so let‚Äôs choose zero for this example. The recursive definition is therefore
    \begin{align*} a_0 \amp =2\\ a_n \amp = a_{n-1}+8 \text{ for }n\ge 1 \end{align*}
    To generalize, the recursive formula for any arithmetic sequence is
    \begin{align*} a_m\amp = \lt\text{insert value here}\gt\\ a_n\amp = a_{n-1}+d \text{ for }n\ge m+1 \end{align*}

    Subsection 3.2.3 General Formulae for Arithmetic Sequences

    Let‚Äôs examine the previous example in more detail to see if we can recognize any patterns and come up with a general formula. Rewriting each term, we get
    \begin{equation*} \begin{array}{*{20}{c}} {2,}\amp{10,}\amp{18,}\amp{26,}\amp{\ldots}\\ 2\amp{2 + 8,}\amp{2 + 8\times 2,}\amp{2 + 8\times 3,}\amp{\ldots} \end{array} \end{equation*}
    So the \(\uprd{3}\) term equals the first plus 8 times 2, the \(\upth{4}\) term equals the first plus 8 times 3, and the \(\nth{}\) term will equal the first plus 8 times \((n - 1)\text{.}\) In other words,
    \begin{equation*} \begin{array}{*{20}{c}} {2,}\amp{10,}\amp{18,}\amp{26,}\amp{\ldots}\amp{a_n}\\ 2\amp{2 + 8,}\amp{2 + 8 \times 2,}\amp{2 + 8 \times 3,}\amp{\ldots}\amp{2+8 \times (n-1)} \end{array} \end{equation*}
    and so we find for this particular sequence, \(a_n=2+8 \times (n-1)\text{,}\) which simplifies to \(a_n=8n-6\text{.}\)
    We can generalize this formula: the \(\nth{}\) term will equal the first plus \(d\) times \((n - m)\text{,}\) so
    \begin{equation*} a_n = a_m + \left( n - m \right)d \text{ where } n \ge m \end{equation*}
    for any arithmetic sequence.

    Example 3.2.4.

    Write a general formula for the sequence \(3, 8, 13, 18, \ldots\text{.}\)
    Answer.
    This sequence is arithmetic with the first term 3 and common difference 5. Let‚Äôs use a starting index of zero.
    \begin{align*} a_n \amp = a_m + \left( n - m \right)d\\ \amp = {a_0} + \left( {n - 0} \right)d\\ \amp = 3 + \left( {n} \right)5\\ \amp = 3 + 5n\\ \amp = 5n + 3 \end{align*}
    The general formula is then \(a_n = 5n + 3\) for \(n \ge 0\text{.}\)

    Example 3.2.5.

    What is the \(\upth{50}\) term in the sequence in the sequence \(3, 8, 13, 18, \ldots\text{?}\)
    Answer.
    This is the same sequence from the previous example. We may then use the formula we derived, \(a_n = 5n + 3\) for \(n \ge 0\text{.}\) But we do have to be careful about our index \(n\text{.}\) Recalling that the number of terms \(k\) is given by
    \begin{equation*} k=n-m+1 \end{equation*}
    where we used \(m = 0\) as our starting index, then
    \begin{align*} 50 \amp = n-0=1\\ n\amp = 49 \end{align*}
    and so the \(\upth{50}\) term will be \(a_{49}\text{.}\)
    \begin{align*} a_n \amp = 5n + 3\\ a_{49} \amp = 5 \times 49 + 3\\ \amp = 245 + 3\\ \amp =248 \end{align*}
    The \(\upth{50}\) term is 248.

    Example 3.2.6.

    What is the common difference in the arithmetic sequence in which the first term is 18 and the twelfth term is \(-59\text{?}\)
    Answer.
    The easiest way to count these terms correctly is to have a starting index of one, and then the first term is \(a_1\) and the twelfth term is \(a_{12}\text{.}\)
    \begin{align*} a_n \amp = a_m + \left( n - m \right)d\\ a_{12} \amp = a_1 + \left( 12 - 1 \right)d\\ -59 \amp = 18 + \left( 12 - 1 \right)d\\ -77 \amp = 11d\\ d\amp = -7 \end{align*}
    The common difference is \(-7\text{.}\)

    Example 3.2.7.

    Which term has a value of 404 in the sequence \(-37, -28, -19, \ldots\text{?}\)
    Answer.
    Let‚Äôs use a starting index of one. So \(a_1\) is \(-37\) and \(d\) is \(+9\text{.}\) Then we want to find the value of \(n\) for which \(a_n\) equals 404.
    \begin{align*} a_n \amp = a_m + \left( n - m \right)d\\ \amp = a_1 + \left(n - 1 \right)d\\ 404 \amp = - 37 + \left( n - 1 \right)9\\ 441 \amp = 9\left( n - 1 \right)\\ 49 \amp = n - 1\\ n \amp = 50 \end{align*}
    The fiftieth term is 404.

    Subsection 3.2.4 Arithmetic Series

    Recall that \(S_k\) is the sum of the first \(k\) terms of a series. Let‚Äôs look at a couple of examples of arithmetic series to see if we can identify any patterns.
    Suppose we wish to take some partial sums of the series \(2 + 10 + 18 + 26 + \ldots\text{.}\) Let‚Äôs first calculate \(S_6\text{.}\) We could just find the first six terms and add them up, but notice the following:
    \begin{equation*} S_6 = 2 + 10 + 18 + 26 + 34 + 42 \end{equation*}
    The sum of the first and last numbers is 44. The sum of the second and second-to-last is also 44. So is the sum of the third and third-last. So when you take the terms in pairs, each pair has the same sum, \((a_m + a_n)\text{,}\) and there are \(k/2\) pairs in total. Then
    \begin{equation*} S_k = \frac{k}{2}\left( a_m + a_n \right) \end{equation*}
    What if, however, there are an odd number of terms? Let‚Äôs also calculate \(S_7\text{:}\)
    \begin{equation*} S_7 = 2 + 10 + 18 + 26 + 34 + 42 + 50 \end{equation*}
    The sum of the first and last is 52, as is the sum of the each ‚Äúinner pair‚Äù. Notice that the middle, unpaired value, is \(\frac{1}{2}\) of 52. So in a sense, the middle term is \(\frac{1}{2}\) of a pair, for a total of \(3\frac{1}{2}\) pairs. But that‚Äôs just \(7/2\text{,}\) which is our \(k/2\) in the original formula! So we‚Äôre still good. The relationship
    \begin{equation*} S_k = \frac{k}{2}\left( a_m + a_n \right) \end{equation*}
    still works, for both odd and even values of \(k\text{.}\)
    Generalizing, we find that
    \begin{equation*} S_k = \frac{k}{2}\left(a_m + a_n \right) \end{equation*}
    where \(k\) can be even or odd and
    \begin{equation*} k = n - m + 1 \text{ for } n \ge m \end{equation*}

    Example 3.2.8.

    Find the sum of the first forty terms of the series \(2 + 10 + 18 + 26 + \ldots\text{.}\)
    Answer.
    This is just the same sequence as before, with first term \(a_m = 2\) and common difference \(d = 8\text{.}\) In order to use our previous formula, however, we need to calculate the last term \(a_n\text{.}\) If we start with an index of one, then the fortieth term will be \(a_{40}\text{,}\) and we will need that value to calculate \(S_{40}\text{.}\)
    \begin{align*} a_n \amp = a_m + \left( n - m \right)d\\ \amp = a_1 + \left( n - 1 \right)d\\ a_{40} \amp = 2 + 39 \times 8\\ \amp = 314 \end{align*}
    So,
    \begin{align*} S_k \amp = \frac{k}{2}\left( a_m + a_n \right)\\ S_{40} \amp = \frac{40}{2}\left( 2 + 314 \right)\\ \amp = 20 \times 316\\ \amp = 6320 \end{align*}
    The sum of the first forty terms is 6320. (Much easier than writing out the first forty terms and adding them up!)
    In the previous example, we used the formula for \(a_n\) to calculate the last term and put its value into the formula for \(S_k\text{.}\) We could do that in a more general way:
    \begin{align*} S_k \amp = \frac{k}{2}\left(a_m + a_n \right)\\ \amp = \frac{k}{2}\left[ a_m + \left(a_m + \left(n - m \right)d \right) \right]\\ \amp = \frac{k}{2}\left[ 2a_m + \left( n - m \right)d \right] \end{align*}
    The last expression, which gives \(S_k\) as a function of the first term, the number of terms, and the common difference, can also be used to evaluate series.

    Example 3.2.9.

    Find the sum of the first one hundred terms of the sequence \(5, -6, -17, -26, \ldots\text{.}\)
    Answer.
    This sum will just be \(5 + -6 + -17 + -26 + \ldots\text{,}\) with \(a_m = 5\text{,}\) \(d = -11\text{,}\) and \(k = 100\text{.}\) If we start our index at one, then
    \begin{align*} k \amp = n - m + 1\\ 100 \amp = n - 1 + 1\\ n\amp = 100 \end{align*}
    and we can substitute this into the equation for \(S_k\text{:}\)
    \begin{align*} S_k \amp = \frac{k}{2}\left[ 2a_m + \left( n - m \right)d \right]\\ S_{100} \amp = \frac{100}{2}\left[ 2 \times 5 + 99 \times \left( - 11 \right) \right]\\ \amp = - 53950 \end{align*}

    Example 3.2.10.

    Calculate \(\sum\limits_{j = 3}^{18} {5n + 10}\text{.}\)
    Answer.
    The first term will be for \(j=3\) and will equal \(5(3)+10=25\text{.}\) Next is \(j=4\) and will equal \(5(4)+10=30\text{,}\) \(j=5\) equaling \(5(5)=35\text{,}\) and so on. The last term will be for \(j=18\) and will equal \(5(18)+10=100\text{.}\)
    In other words, our series is \(25+30+35+\ldots 100\text{.}\) Is it arithmetic? Yes, with common difference \(d = 5\text{.}\)
    What else do we need for our calculation? The number of terms is
    \begin{align*} k \amp = n-m+1\\ k \amp = 18-3+1\\ k\amp = 16 \end{align*}
    Then
    \begin{align*} S_n \amp = \frac{n}{2}\left(a_m + a_n \right)\\ S_{16} \amp = \frac{16}{2}\left(25 + 100 \right) = 1000 \end{align*}

    Example 3.2.11.

    Pat the math instructor asks her students to do five word problems the first week, six the second week, seven the third week, and so on, increasing the number of word problems each week by one.
    1. How many word problems will diligent students be doing in the last week of classes (the \(\nth{11}\) week)?
    2. How many word problems will diligent students have completed during the course of the term (11 weeks)?
    Answer.
    1. The number of word problems is a sequence: \(5, 6, 7, \ldots\) . In fact, it‚Äôs an arithmetic sequence with \(a_m = 5\) and \(d = 1\text{.}\) If we start our counting from one, then in the eleventh week,
      \begin{align*} a_n \amp = a_m + \left(n - m \right)d\\ a_n \amp = a_1+\left(n-1\right)d\\ a_{11} \amp = 5+10\times 1\\ \amp = 15 \end{align*}
      Diligent students will solve 15 word problems in the last week of classes.
    2. The total number of word problems solved is
      \begin{align*} S_k \amp = \frac{k}{2}\left(a_m + a_n \right)\\ S_{11} \amp = \frac{11}{2}\left(5 + 15\right)\\ \amp = 110 \end{align*}
    Diligent students will have solved 110 word problems in total.

    Subsection 3.2.5 Summary

    For an arithmetic sequence, the nth term is given by
    \begin{equation*} a_n = a_m + \left(n - m \right)d \text{ for } n \ge m \end{equation*}
    For an arithmetic series, the sum of the first \(k\) terms (\(\kth{}\) partial sum) is
    \begin{equation*} S_k = \frac{k}{2}\left(a_m + a_n \right) \end{equation*}
    or
    \begin{equation*} S_k = \frac{k}{2}\left[ 2a_m + \left(n - m \right)d \right] \end{equation*}
    where \(k=n-m+1\) and \(n\ge m\text{.}\)
