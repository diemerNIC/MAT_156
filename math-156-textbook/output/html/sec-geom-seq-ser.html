Section 3.3 Geometric Sequences and Series

Subsection 3.3.1 Geometric Sequences

Let's start out with a definition:

Definition 3.3.1.

A geometric sequence is a sequence in which the next term is found by multiplying the previous term by a constant (the common ratio\(\ r\))

Here are some examples of geometric sequences:

  1. \(\displaystyle 9, 18, 36, 72, \ldots\)

  2. \(\displaystyle 12, 18, 27, \frac{81}{2},\ldots\)

  3. \(\displaystyle 10, -30, 90, -270, \ldots, -196830\)

  4. \(\displaystyle -3, -12, -48, -192, \ldots\)

  5. \(\displaystyle 48, -36, 27, \ldots\)

The common ratios of each of these sequences, in order from a) to e), is \(2\text{,}\) \(\frac{3}{2}\text{,}\) \(-3\text{,}\) \(4\text{,}\) \(-\frac{3}{4}\text{,}\) respectively. Note that in each of them, we can find the common ratio \(r\) by taking \(any\) term and dividing it by the previous term.

Like any other sequences, geometric sequences can be finite or infinite. Example c) above is finite, as the last term is specified. The others are infinite sequences.

Example 3.3.2.

For each of the following sequences, state whether it is arithmetic, geometric, or neither.

  1. \(\displaystyle 45, 15, 5, \ldots\)

  2. \(\displaystyle 5, 3, 1, -1, \ldots\)

  3. \(\displaystyle 1, 8, 27, 64, \ldots , 1000\)

  4. \(\displaystyle -1, 1, -1, 1, -1, 1, \ldots\)

Answer.

  1. Geometric, because the common ratio \(r\) is \(\frac{1}{3}\)

  2. Arithmetic, because the common difference \(d\) is \(-2\text{.}\)

  3. Neither, because there isn't either a common difference or ratio between terms. (In fact, the pattern is that \(a_n = n^3\) for \(n \ge 1\text{.}\))

  4. Geometric, because the common ratio \(r\) is \(-1\text{.}\)

Again, you can define a geometric sequence in one of three ways: by listing the terms, by giving a recursive definition, or by giving a general definition.

Subsection 3.3.2 Recursive Definitions for Geometric Sequences

Let's look at an example.

Example 3.3.3.

Give a recursive definition for the sequence \(2, 10, 50, 250, \ldots\)
Answer.
Recall that a recursive definition has two parts: listing the first term and giving the pattern. In this case, the pattern is multiplying the previous term by \(r = 5\) to get the next term. Let's use 0 as our starting index. The recursive definition is therefore
\begin{align*} a_0 \amp = 2\\ a_n \amp = 5a_{n - 1} \text{ for } n \ge 1 \end{align*}

Generally, the recursive definition for any geometric sequence is

\begin{align*} a_m \amp = \lt \text{ insert value here } \gt\\ a_n \amp = r\times a_{n-1},\, n \ge m+1 \end{align*}

Subsection 3.3.3 General Formulae for Geometric Sequences

Let's examine the previous example in more detail to see if we can recognize any patterns and come up with a general formula. Rewriting each term, we get

\begin{equation*} \begin{array}{*{20}{c}} 2,\amp 10,\amp 50,\amp 250,\amp \ldots\\ 2,\amp 2 \times 5,\amp 2 \times {5^2},\amp 2 \times {5^3},\amp \ldots\\ \end{array} \end{equation*}

So the \(\uprd{3}\) term equals the first times 5 squared, the \(\upth{4}\) term equals the first times 5 cubed, and the \(\nth{}\) term will equal the first times 5 raised to the \((n - 1)\) power. In general, for sequences with first term \(a_m\text{,}\) the \(nth{}\) term equals the first term times \(r\) raised to the \((n - m)\) power, namely

\begin{equation*} a_n=a_m r^{n-m} \end{equation*}

for all geometric sequences.

Example 3.3.4.

Write a general formula for the sequence \(3, 6, 12, \ldots\)
Answer.
This sequence is geometric with the first term 3 and common ratio 2. If we choose \(n = 1\) for our first term, then
\begin{align*} a_n \amp = a_m r^{n-m}\\ \amp = a_1 r^{n-1}\\ \amp = 3\times\left(2^{n-1}\right) \end{align*}
The general formula is then \(a_3\times 2^{n-1}\) for \(n\ge 1\text{.}\)

Example 3.3.5.

What is the \(\upth{20}\) term in the sequence in the sequence \(3, 6, 12, \ldots\text{?}\)
Answer.
This is the same sequence from the previous example. We may then use the formula we derived above with \(n = 20\text{.}\)
\begin{align*} a_n\amp = a_m r^{n-m}\\ \amp = a_1 r^{n-1}\\ a_{20}\amp = 3\times 2^{20-1}\\ \amp = 3\times 2^{19}\\ a_{20}=1,572,864 \end{align*}
The \(\upth{20}\) term is 1,572,864, which provides a nice example for how fast geometric sequences can grow, even for small values of \(r\text{.}\)

Example 3.3.6.

Write a general formula for the sequence \(8, 12, 18, 27, \ldots\) What is the fifteenth term in this sequence? The fiftieth?
Answer.
If we start our counting at \(n=1\text{,}\) then the fifteenth term is \(a_{15}\) and the fiftieth term is \(a_{50}\text{.}\)
\begin{align*} a_n\amp = a_m r^{n-m}\\ \amp = a_1 r^{n-1}\\ a_n \amp = 8\left(\frac{3}{2}\right)^{n-1}\\ a_{15}\amp = 8\left(\frac{3}{2}\right)^{14}\approx 2335.43\\ a_{50}\amp = 8\left(\frac{3}{2}\right)^{50}\approx 3.40065\times 10^9 \end{align*}
So the general formula is \(a_n = 8\left( \frac{3}{2} \right)^{n - 1}\) for \(n \ge 1\) and the fifteenth and fiftieth terms are approximately 2335.43 and \(3.4 \times 10^9\text{,}\) respectively.

Subsection 3.3.4 Geometric Series

Recall that \(S_k\) is the sum of the first \(k\) terms of a series. Let's look at how a formula for \(S_k\) is derived, using a series that starts with \(n=1\text{.}\)

\begin{align*} S_k \amp = \amp{a_1} \amp+ \amp{a_2} \amp+ \amp{a_3} \amp+ \amp{a_4} \amp+ \amp\ldots \amp+ \amp{a_{n - 2}} \amp+ \amp{a_{n - 1}} \amp+ \amp{a_k}\\ S_k \amp = \amp{a_1} \amp+ \amp{a_1}r \amp+ \amp{a_1}{r^2} \amp+ \amp{a_1}{r^3} \amp+ \amp\ldots \amp+ \amp{a_1}{r^{k - 3}} \amp+ \amp{a_1}{r^{k - 2}} \amp+ \amp{a_1}{r^{k - 1}} \end{align*}

Let's take that last expression for \(S_k\) and multiply it by \(-r\) to get

\begin{equation*} - r{S_k} = - {a_1}r - {a_1}{r^2} - {a_1}{r^3} - {a_1}{r^4} - ... - {a_1}{r^{k - 2}} - {a_1}{r^{k - 1}} - {a_1}{r^k} \end{equation*}

Then if we add the rows for \(S_k\) and \(-rS_k\text{,}\) we get

\begin{align*} {S_k} \amp= \amp {a_1} \amp+ {a_1}r \amp+ {a_1}{r^2} \amp+ {a_1}{r^3} \amp+ \ldots \amp+ {a_1}{r^{k - 3}} \amp+ {a_1}{r^{k - 2}} \amp+ {a_1}{r^{k - 1}}\\ - r{S_k} \amp=\amp\amp - {a_1}r \amp- {a_1}{r^2} \amp- {a_1}{r^3} \amp- {a_1}{r^4} \amp- \ldots \amp- {a_1}{r^{k - 2}} \amp- {a_1}{r^{k - 1}} \amp- {a_1}{r^k}\\ \hline\\ {S_k} - r{S_k} \amp=\amp {a_1} \amp\amp\amp\amp\amp\amp\amp\amp- {a_1}{r^k} \end{align*}

since all of the terms in between these two (\(a_1\) and \(a_1r^k\)) combine pair-wise to zero by subtraction. Then

\begin{align*} {S_k} - r{S_k}\amp = {a_1}\left( {1 - {r^k}} \right)\\ {S_k}\left( {1 - r} \right) \amp = {a_1}\left( {1 - {r^k}} \right) \end{align*}

and so

\begin{equation*} {S_k} = \frac{{{a_1}\left( {1 - {r^k}} \right)}}{{\left( {1 - r} \right)}} \end{equation*}

To generalize, the formula for the sum of the first \(n\) terms for any geometric series that starts with first term \(a_m\) is

\begin{equation*} {S_k} = \frac{{{a_m}\left( {1 - {r^k}} \right)}}{{\left( {1 - r} \right)}} \end{equation*}

Example 3.3.7.

Find the sum of the first 20 terms of the series \(3 + 6 + 12 + \ldots\)
Answer.
This is a geometric series with \(a_m = 3\) and \(r = 2\text{.}\) We want to find \(S_{20}\text{.}\)
\begin{align*} {S_k} \amp = \frac{{{a_m}\left( {1 - {r^k}} \right)}}{{\left( {1 - r} \right)}}\\ {S_{20}} \amp = \frac{{3\left( {1 - {2^{20}}} \right)}}{{\left( {1 - 2} \right)}} = 3,145,725 \end{align*}
The sum of the first 20 terms is 3,145,725.

Example 3.3.8.

Find the sum of the first forty terms of the series \(8 - 12 + 18 - 27 \ldots\text{.}\)
Answer.
This is a geometric series with \(a_m = 8\) and \(r = -\frac{3}{2}\text{.}\) We want to find \(S_{40}\text{.}\)
\begin{align*} {S_k} \amp = \frac{{{a_m}\left( {1 - {r^k}} \right)}}{{\left( {1 - r} \right)}}\\ {S_{20}} \amp = \frac{{8\left( {1 - {{\left( { - 1.5} \right)}^{40}}} \right)}}{{\left( {1 - \left( { - 1.5} \right)} \right)}} = - 3.53835 \times {10^7} \end{align*}
The sum of the first forty terms is approximately \(-3.54 \times{} 10^7\text{.}\)

Subsection 3.3.5 Sum of an Infinite Geometric Series

Let's take a look at the infinite series \(\frac{1}{2} + \frac{1}{4} + \frac{1}{8} + \frac{1}{{16}} + \ldots\) What happens when we try to evaluate this sum using the \(S_k\) formula? We can put \(a_m = \frac{1}{2}\text{,}\) \(r = \frac{1}{2}\text{,}\) but can we put \(n = \infty\) into the formula? It is important to note that \(\infty\) is not a number. Instead, it is a symbol that represents the characteristic of being unbounded, or “going on forever”.

Let's take a closer look at the behaviour of \((\frac{1}{2})^n\) for large values of \(n\text{.}\) As \(n\) gets larger, the fraction \({\left( {\frac{1}{2}} \right)^n} = \frac{1}{{{2^n}}}\) gets ever smaller. In fact, as \(n\) grows without bound (sometimes written \(n\to\infty\)), \((\frac{1}{2})^n \) will approach zero.

This is true for any \(r\) provided that \(|r| \lt 1\text{.}\) (If you're not familiar with the absolute value bars, an equivalent expression is that \(-1 \lt r \lt 1\text{.}\))

Recalling that

\begin{equation*} {S_k} = \frac{{{a_m}\left( {1 - {r^k}} \right)}}{{\left( {1 - r} \right)}} \end{equation*}

and letting the \(r^n\) term go to zero, then

\begin{equation*} {S_\infty } = \frac{{{a_m}}}{{1 - r}} \text{ for } -1 \lt r \lt 1 \end{equation*}

for any infinite geometric series with \(a_m\) the first term, provided that \(r\) meets the restriction above.

Let's now revisit the series that started this discussion and evaluate it in the following example.

Example 3.3.9.

Evaluate \(\frac{1}{2} + \frac{1}{4} + \frac{1}{8} + \frac{1}{{16}} +\ldots\)
Answer.
This series is geometric with \(a_m = \frac{1}{2}\) and \(r = \frac{1}{2}\text{.}\) Then
\begin{equation*} {S_\infty } = \frac{{{a_m}}}{{1 - r}} = \frac{{1/2}}{{1 - 1/2}} = \frac{{1/2}}{{1/2}} = 1 \end{equation*}
The sum of this series is 1.

Example 3.3.10.

Evaluate \(24 + 16 + \frac{32}{3}+ \ldots\)
Answer.
This series is geometric with \(a_m = 24\) and \(r = \frac{2}{3}\text{.}\)
\begin{equation*} {S_\infty } = \frac{{{a_m}}}{{1 - r}} = \frac{{24}}{{1 - \frac{2}{3}}} = \frac{{24}}{{\frac{1}{3}}} = 24 \times \frac{3}{1} = 72 \end{equation*}

Example 3.3.11.

Evaluate \(24 - 16 + \frac{32}{3}+ \ldots\)
Answer.
This series is identical to the previous one except that \(r\) is now negative: \(a_m = 24\) and \(r = -\frac{2}{3}\text{.}\)
\begin{equation*} {S_\infty } = \frac{{{a_m}}}{{1 - r}} = \frac{{24}}{{1 - \left( { - \frac{2}{3}} \right)}} = \frac{{24}}{{1 + \frac{2}{3}}} = \frac{{24}}{{\frac{5}{3}}} = 24 \times \frac{3}{5} = \frac{{72}}{5} = 14.4 \end{equation*}

Example 3.3.12.

Evaluate \(12 + 18 + 27 + \ldots\text{.}\)
Answer.
This series is geometric with \(a_m = 12\) and \(r = \frac{3}{2}\text{.}\) You may already realize what's going on, but in case you don't, let's naively put the values into the formula and see what we get:
\begin{equation*} {S_\infty } = \frac{{{a_m}}}{{1 - r}} = \frac{{12}}{{1 - \frac{3}{2}}} = \frac{{12}}{{ - \frac{1}{2}}} = 12 \times - \frac{2}{1} = - 24 \end{equation*}
Wait! How can the sum of a bunch of positive number be negative? The answer is that our restriction for \(r\) is that it must be between \(-1\) and 1, but \(r = 1.5\text{.}\) Because \(r\) does not satisfy the restriction, we cannot use the above formula for \(S_\infty\text{.}\) Indeed, if you add up a bunch of positive numbers that are increasing as you go up, you can see that the sum just keeps getting bigger as we add more terms. You could then either say that the sum is infinite or “does not exist”. These two statements are not quite equivalent. While it is correct to say that the sum does not exist, it is more descriptive to state that the sum is infinite (or to write “\(=+\infty\)”). There could be a sum that grows without bound “in the negative direction”, in which case it would be more descriptive to write “\(=-\infty\)” than simply to say that the sum does not exist.

Let's look at three specific cases:

  1. \(\displaystyle 12+18+27+\ldots\)

  2. \(\displaystyle -12-18-27-\ldots\)

  3. \(\displaystyle 12-18+27+\ldots\)

Each term in (a) is getting more positive, so the sum of that sequence does not exist but writing “\(=+\infty\)” is more descriptive. Each term in (b) is getting more and more negative, so the sum of that sequence does not exist, but writing “\(=-\infty\)” is more descriptive. But in the last case, the sum oscillates back and forth: \(S_1 = 12\text{,}\) \(S_2 = -6\text{,}\) \(S_3 = 21\text{,}\) \(S_4 = -19.5\text{,}\) and so on. The sign of \(S_k\) is either positive or negative depending on whether the number of terms you've added is even or odd. In this case, all we can say is that the sum does not exist.

Example 3.3.13.

Evaluate \(\sum\limits_{j = 0}^\infty {27{{\left( {\frac{1}{3}} \right)}^j}}\text{.}\)
Answer.
The best place to start is to figure out the first few terms to determine the pattern:
\begin{align*} \text{when } j = 0,\quad \amp 27{\left( {\frac{1}{3}} \right)^0} = 27 \times 1 = 27\\ \text{when } j = 1,\quad \amp 27{\left( {\frac{1}{3}} \right)^1} = 27 \times \frac{1}{3} = 9\\ \text{when } j = 2,\quad \amp 27{\left( {\frac{1}{3}} \right)^2} = 27 \times \frac{1}{{{3^2}}} = 3 \end{align*}
so our sequence is \(27, 9, 3, \ldots\) This is geometric with \(a_m = 27\) and \(r = \frac{1}{3}\text{.}\) Then
\begin{equation*} {S_\infty } = \frac{{{a_m}}}{{1 - r}} = \frac{{27}}{{1 - \frac{1}{3}}} = \frac{{27}}{{\frac{2}{3}}} = 27 \times \frac{3}{2} = \frac{{81}}{2} = 40.5 \end{equation*}

Example 3.3.14.

Evaluate \(\sum\limits_{k = 5}^\infty {\;\frac{1}{2}} k\text{.}\)
Answer.
As in the previous example, let's figure out the first few terms to determine the pattern:
\begin{align*} \text{when } k = 5,\quad \amp \frac{1}{2}k = \frac{1}{2}5 = 2.5\\ \text{when } k = 6,\quad \amp \frac{1}{2}k = \frac{1}{2}6 = 3\\ \text{when } k = 7,\quad \amp \frac{1}{2}k = \frac{1}{2}7 = 3.5 \end{align*}
so our sequence is \(2.5, 3, 3.5, \ldots\text{.}\) Wait! This sequence is arithmetic! Not only that, but the numbers are increasing. So the sum of the series does not exist due to unbounded growth in the positive direction. You can write “does not exist” or, if you prefer, the more descriptive “\(=+\infty\)”.

Subsection 3.3.6 Repeating Decimals

Let's examine \(0.\overline 7\) in some detail to see what we find:

\begin{align*} 0.\overline 7 \amp = 0.777777777...\\ \amp = 0.7 + 0.07 + 0.007 + 0.0007 + \ldots \end{align*}

But this is just the sum of an infinite series with \(a_m = 0.7\) and \(r = 0.1\text{.}\) Rewriting \(a_1\) and \(r\) in fraction form (you'll see why in a minute) gives \(a_m = \frac{7}{10}\) and \(r =\frac{1}{10}\text{.}\) Then

\begin{equation*} {S_\infty } = \frac{{{a_m}}}{{1 - r}} = \frac{{\frac{7}{{10}}}}{{1 - \frac{1}{{10}}}} = \frac{{\frac{7}{{10}}}}{{\frac{9}{{10}}}} = \frac{7}{{10}} \times \frac{{10}}{9} = \frac{7}{9} \end{equation*}

So \(0.\overline 7 = 7/9\text{.}\) Interesting!

Example 3.3.15.

Find an exact fraction for \(0.\overline 6 \text{.}\)
Answer.
\begin{align*} 0.\overline 6 \amp = 0.66666666\ldots\\ \amp = 0.6 + 0.06 + 0.006 + 0.0006 + \ldots \end{align*}
But this is just the sum of an infinite geometric series with \(a_m = \frac{6}{10}\) and \(r = \frac{1}{10}\text{.}\) Then
\begin{equation*} {S_\infty } = \frac{{{a_m}}}{{1 - r}} = \frac{{\frac{6}{{10}}}}{{1 - \frac{1}{{10}}}} = \frac{{\frac{6}{{10}}}}{{\frac{9}{{10}}}} = \frac{6}{{10}} \times \frac{{10}}{9} = \frac{6}{9} = \frac{2}{3} \end{equation*}
So \(0.\overline 6 = 2/3\text{.}\)

Example 3.3.16.

Find an exact fraction for \(0.\overline {18}\text{.}\)
Answer.
\begin{align*} 0.\overline {18} \amp = 0.1818181818\ldots\\ \amp = 0.18 + 0.0018 + 0.000018 + \ldots \end{align*}
But this is just the sum of an infinite geometric series with \(a_m = \frac{18}{100}\) and \(r = \frac{1}{100}\text{.}\) Then
\begin{equation*} {S_\infty } = \frac{{{a_m}}}{{1 - r}} = \frac{{\frac{{18}}{{100}}}}{{1 - \frac{1}{{100}}}} = \frac{{\frac{{18}}{{100}}}}{{\frac{{99}}{{100}}}} = \frac{{18}}{{100}} \times \frac{{100}}{{99}} = \frac{{18}}{{99}} = \frac{2}{{11}} \end{equation*}
So \(0.\overline {18} = 2/11\text{.}\)

Subsection 3.3.7 Summary

For a geometric sequence, the \(\nth{}\) term is given by

\begin{equation*} {a_n} = {a_m}{r^{n - m}} \end{equation*}

for \(n \ge m\)

For a geometric series, the sum of the first \(k\) terms (\(\kth{}\) partial sum) is

\begin{equation*} {S_k} = \frac{{{a_m}\left( {1 - {r^k}} \right)}}{{\left( {1 - r} \right)}} \end{equation*}

where \(a_m\) is the first term.

For an infinite geometric series, the sum is

\begin{equation*} {S_\infty } = \frac{{{a_m}}}{{1 - r}} \end{equation*}

provided that \(-1 \lt r \lt 1\) and \(a_m\) is the first term.
