    Section 2.5 Laws of Logic

    You may have noticed some common patterns running through some of the exercises by now. Let‚Äôs examine those patterns in more detail.
    First, let us look at the connections between the two sets of symbols we‚Äôve used so far.
    Logic \(p{\wedge} q\) \(p{\vee} q\) \(\sim\!{p}\) \(F\) \(T\)
    Boolean Algebra \(AB\) \(A+B\) \(\overline{A}\) 0 1
    In each case, we have symbols for negation, or, and and. There are also equivalences with False/True for logic and 0/1 (or off/on) for Boolean algebra and logic circuits. Let‚Äôs see what else they have in common.

    Subsection 2.5.1 Identity Laws

    Consider the following truth table for the logical symbols.
    \(p\) \(0\) \(1\) \(p{\wedge} 0\) \(p{\vee} 0\) \(p{\wedge} 1\) \(p{\vee} 1\)
    0 0 1 0 0 0 1
    1 0 1 0 1 1 1
    You can replace the logic symbols in this table with the corresponding Boolean algebra notation. From this, we can deduce the following identity laws:

    Identity Laws.

    Logic
    \begin{align*} p{\wedge} 0 \amp\Leftrightarrow 0\\ p{\vee} 0\amp\Leftrightarrow p\\ p{\wedge} 1\amp\Leftrightarrow p\\ p{\vee} 1\amp\Leftrightarrow 1 \end{align*}
    Boolean
    \begin{align*} A\cdot 0\amp = 0\\ A+0\amp = A\\ A\cdot 1\amp = A\\ A+1\amp = 1 \end{align*}
    It is interesting to note that the first three Boolean identity laws look like laws of normal arithmetic with numbers. The fourth law does not resemble any arithmetic law. This is a clue that Boolean algebra is not equivalent to the algebra associated with the real numbers.

    Subsection 2.5.2 Idempotent Laws

    We deduce the idempotent laws in the same way. Consider the following table, given using logic notation:
    \(p\) \(p{\wedge} p\) \(p{\vee} p\)
    0 0 0
    1 1 1
    Boolean algebra notation could have been used in this table, instead.
    From this we can deduce the following:

    Idempotent Laws.

    Logic
    \begin{align*} p{\wedge} p \amp\Leftrightarrow p\\ p{\vee} p\amp\Leftrightarrow p \end{align*}
    Boolean
    \begin{align*} A\cdot A\amp = A\\ A+A\amp = A \end{align*}

    Subsection 2.5.3 Complement Laws

    Similarly, truth tables can be used to deduce the complement laws:

    Complement Laws.

    Logic
    \begin{align*} \sim\!(\sim\!{p}) \amp\Leftrightarrow p\\ p\,{\wedge}\,\sim\!{p}\amp\Leftrightarrow 0\\ p\,{\vee}\,\sim\!{p}\amp\Leftrightarrow 1 \end{align*}
    Boolean
    \begin{align*} \overline{\overline{A}}\amp =A\\ A\cdot\overline{A}\amp = 0\\ A+\overline{A}\amp = 1 \end{align*}

    Subsection 2.5.4 Commutative Laws

    The commutative laws state that the ordering of symbols is not significant for the logic operations and and or, and also for the corresponding operations in Boolean algebra. The commutative law does not apply to all operations. More on that later.

    Commutative Laws.

    Logic
    \begin{align*} p{\wedge} q \amp\Leftrightarrow q{\wedge} p\\ p {\vee} q\amp\Leftrightarrow q{\vee} p \end{align*}
    Boolean
    \begin{align*} A\cdot B\amp =B\cdot A\\ A+B\amp =B+A \end{align*}

    Subsection 2.5.5 Associative Laws

    Exercise¬†2.3.3.16 illustrates the associative law for or. You could construct a similar truth table for and. The associative laws in Boolean algebra form look just like the associative laws of multiplication and addition of real numbers. Remember, though, that Boolean algebra is not the same as the algebra of real numbers!

    Associative Laws.

    Logic
    \begin{align*} (p{\wedge} q){\wedge} r \amp\Leftrightarrow p{\wedge}(q{\wedge} r)\\ (p{\vee} q){\vee} r\amp\Leftrightarrow p{\vee}(q{\vee} r) \end{align*}
    Boolean
    \begin{align*} (A\cdot B)\cdot C\amp = A\cdot(B\cdot C)\\ (A+B)+C\amp =A+(B+C) \end{align*}

    Subsection 2.5.6 Summary So Far

    Here are all the laws identified so far, collected into one table for easy reference. In the next subsection, we will examine how these laws are used to simplify logical/Boolean expressions without having to resort to constructing truth tables.
    Law Logic Boolean Algebra
    Identity \(p{\wedge} 1\Leftrightarrow p\) \(A\cdot 1=A\)
    \(p{\vee} 1\Leftrightarrow 1\) \(A+1=1\)
    \(p{\wedge} 0\Leftrightarrow 0\) \(A\cdot 0=0\)
    \(p{\vee} 0\Leftrightarrow p\) \(A+0=A\)
    Idempotent \(p{\wedge} p\Leftrightarrow p\) \(A\cdot A=A\)
    \(p{\vee} p\Leftrightarrow p\) \(A+A=A\)
    Complement \(\sim\!(\sim\!{p})\Leftrightarrow p\) \(\overline{\overline{A}}=A\)
    \(p\,{\wedge}\,\sim\!{p}\Leftrightarrow 0 \) \(A\cdot\overline{A}=0\)
    \(p\,{\vee}\,\sim\!{p}\Leftrightarrow 1\) \(A+\overline{A}=1\)
    Commutative \(p{\wedge} q\Leftrightarrow q{\wedge} p\) \(A\cdot B=B\cdot A\)
    \(p{\vee} q\Leftrightarrow q{\vee} p\) \(A+B=B+A\)
    Associative \((p{\wedge} q){\wedge} r\Leftrightarrow p{\wedge}(q{\wedge} r)\) \((A\cdot B)\cdot C=A\cdot(B\cdot C)\)
    \((p{\vee} q){\vee} r\Leftrightarrow p{\vee}(q{\vee} r)\) \((A+B)+C=A+(B+C)\)

    Subsection 2.5.7 Simplifying Logical Expressions

    We use the laws of logic to simplify logical expressions or to prove logical equivalence without resorting to truth tables.
    Suppose we wish to simplify \((p{\wedge} q){\vee}(\sim\!{q}\,{\wedge}\, 0){\vee}(\sim\!{r}\,{\wedge}\,r)\text{.}\) Note that this would require a truth table with 8 rows to show all combinations of \(p\text{,}\) \(q\text{,}\) and \(r\text{.}\) Less work is required to simplify the expression using the laws of logic. You can think of the laws of logic as summary statements of truth tables.
    The procedure for simplifying an expression using the laws of logic is to simplify each piece of the expression using a single law, then write the name of the law you are using to one side. Writing the name of the law is required, and not optional! If you are using more than one law, then use a separate line for each law/step.
    Let‚Äôs use the laws of logic to simplify the statement given earlier:
    \((p{\wedge} 1){\vee}(\sim\!{q}\,{\wedge}\, 0){\vee}(\sim\!{r}\,{\wedge} r)\)
    \(p{\vee} 0{\vee}(\sim\!{r}\,{\wedge}\,r)\) identity
    \((p{\vee} 0){\vee}(\sim\!{r}\,{\wedge}\,r)\) associative
    \(p{\vee}(\sim\!{r}\,{\wedge}\,r)\) identity
    \(p{\vee}(r\,{\wedge}\,\sim\!{r})\) commutative
    \(p{\vee} 0\) complement
    \(p\) identity
    So we see that \((p{\wedge} q){\vee}(\sim\!{q}\,{\wedge}\, 0){\vee}(\sim\!{r}\,{\wedge}\,r)\Leftrightarrow p\text{.}\)
    The simplification can be done another way:
    \((p{\wedge} 1){\vee}(\sim\!{q}\,{\wedge} \,0){\vee}(\sim\!{r}\,{\wedge}\,r)\)
    \(p{\vee} 0{\vee}(\sim\!{r}\,{\wedge}\,r)\) identity
    \(p{\vee} 0{\vee} 0\) complement
    \(p{\vee} 0\) definition of or
    \(p\) identity
    The conclusion is the same, as we should expect!

    Example 2.5.1.

    Simplify \((p\,{\vee}\,\sim\!{p}){\wedge}(\sim\!{p}\,{\vee}\,\sim\!{p})\text{.}\)
    Answer.
    \((p\,{\vee}\sim\!{p}){\wedge}(\sim\!{p}\,{\vee}\sim\!{p})\)
    \(1{\wedge}(\sim\!{p}\,{\vee}\,\sim\!{p})\) complement
    \(1{\wedge}(\sim\!{p})\) idempotent
    \(\sim\!{p}\) identity
    (If you applied the laws correctly but in a different order or combination, you should still come to the same, correct conclusion.)
